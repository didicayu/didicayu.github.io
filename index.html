My Projects

Doodle Recognition Neural Network

    Made with:

This was my first project using a neural network library that I built from scratch in unity with C#. This demo lets you draw on a canvas whatever you want, and the neural network will try to classify your drawing as either a cat, a rainbow, an airplane or a bannana. You can chose how much do you want to train the network and see how the number of epochs impacts it's performance. This AI was trained using google's "Quick, Draw!" data.

Flappy Bird NeuroEvolution

    Made with:

For this project I used my custom Neural Network library once more. I focused on learning alternative ways of training AI models other than the classic supervised learning, this demo demonstrates how neuro evolution training works. Some updates to the library were made such as being able to store "the brain" on a file
